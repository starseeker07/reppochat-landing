Google Play에서 다운로드

Android 9.0 이상 · 무료 설치

대화는 외부로 나가지 않아요

AI 추론도, 대화 기록도, 첨부한 사진의 인식도 — 모두 모델 파일을 받은 뒤 기기 안에서 처리됩니다. 메시지 본문은 서버로 전송되지 않아요.

기기 안에서 처리

모든 대화는 인터넷 연결 없이 기기 안에서 처리돼요. 추론과 대화 기록은 외부 서버나 클라우드로 전송되지 않아요.

서버로 보내지 않음

대화 본문·프롬프트·AI 응답·페르소나 설정·기억 내용은 운영자 서버나 외부 AI API로 전송되지 않아요.

비행기 모드에서도

모델 파일을 한 번 받아두면, 비행기 모드에서도 대화 생성과 기록 조회가 그대로 동작해요.

이용 방법

세 단계면 충분해요

1

AI 모델을 한 번 받으세요

오픈 소스 언어 모델 기반의 모델 파일을 하나 받으면 준비 끝. 일반 사양과 고사양 중 고를 수 있어요. (Wi-Fi 권장)

2

캐릭터를 고르거나 만들어요

기본 프리셋으로 시작하거나, 이름·성격·말투·배경까지 나만의 페르소나를 원하는 만큼 직접 만들 수 있어요.

3

이제 오프라인으로 대화하세요

인터넷 없이 기기 안에서 바로 대화. 대화를 거듭할수록 친밀도가 오르고, 중요한 기억이 쌓여가요.

주요 기능

레포챗이 특별한 이유

기기 안에서 동작하는 로컬 AI

오픈 소스 언어 모델 기반의 추론 엔진이 기기 안에서 직접 답을 생성합니다. 클라우드도, 외부 API도 거치지 않아요.

  • 일반 사양·고사양 모델 중 선택
  • 여러 모델을 받아 함께 보관
  • 다운로드 후에는 완전 오프라인

나만의 캐릭터를 만들어요

이름, 성격, 말투, 좋아하는 것과 싫어하는 것, 배경 서사까지 직접 설정할 수 있어요. 기본 프리셋으로 시작하거나 원하는 만큼 페르소나를 만들어 보세요.

기억하고 이어가는 대화

대화를 거듭할수록 친밀도가 오르고, 중요한 사실·약속·감정 상태가 장기 기억으로 저장돼요. 핵심 기억은 직접 편집하거나 고정할 수 있어요.

사진을 보여주며 대화해요

사진을 첨부하면 캐릭터가 그 사진을 보고 답해요. 사진 인식도 100% 기기 안에서 이뤄지고, 사진은 외부로 전송되지 않아요. 시스템 사진 선택기를 써서 별도 권한도 필요 없어요.

안심하고 쓰는 보안

로컬 데이터베이스를 암호화하고, PIN과 생체 인증 잠금을 지원해요. 16세 이상 대상으로 세이프티 필터를 적용하고, 메시지 본문은 서버로 전송되지 않아요.

데이터는 직접 옮길 수 있어요

암호화된 .reppo 백업 파일로 다른 기기로 옮기거나 보관할 수 있어요. 백업 비밀번호는 본인만 알고 있으면 누구도 열 수 없어요.

메모리(RAM)

저장 공간
모델당 약 5GB 여유

RAM 6GB 미만 기기는 채팅을 지원하지 않아요. 일반 사양 모델은 RAM 6GB 이상, 고사양 모델은 12GB 이상을 권장하며, 여러 모델을 보관하면 그만큼 저장 공간이 더 필요해요. 사진 인식은 메모리를 많이 써서, 메모리가 적은 기기에서는 느리거나 드물게 앱이 닫힐 수 있어요.
